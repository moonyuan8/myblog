关于本站

本站建于2021年11月，主要是个人爱好写着玩，记录日常生活、学习的技术、科软课程等。
由于最近开始又开始找实习了，比较忙，更新会少一些吧~

  • 前 端 ：Semantic Ui(主要) + Layui
  • 后 端 ：SpringBoot + MYSQL + Redis + Thymeleaf
